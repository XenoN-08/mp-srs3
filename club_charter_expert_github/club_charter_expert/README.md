# 📋 Эксперт по уставу студенческого клуба

Многоагентная система на базе Claude API для анализа и проверки уставов студенческих клубов на соответствие правилам университета.

## 🏗️ Архитектура

```
Загрузка файлов
    │
    ▼
Агент 1 — Извлечение структуры
    │
    ▼
Агент 2 — Проверка соответствия
    │
    ├─── [Нет нарушений] ──────────────────────────────────┐
    │                                                       │
    └─── [Есть нарушения] → Условная задача (доработки)    │
                                        │                   │
                                        ▼                   ▼
                              Агент 3 — Заключение ◄────────┘
                                        │
                                        ▼
                              HITL — Одобрение эксперта
                                        │
                                        ▼
                              Memory ← Сохранение знаний
```

## 🤖 Агенты

| Агент | Задача |
|-------|--------|
| **Агент 1** | Извлекает разделы и метаданные устава |
| **Агент 2** | Проверяет соответствие правилам университета |
| **Условная задача** | Формирует список доработок (только при нарушениях) |
| **Агент 3** | Генерирует экспертное заключение и предложения |

## 🚀 Запуск

```bash
# 1. Клонировать репозиторий
git clone https://github.com/XenoN-08/mp-srs3
cd club-charter-expert

# 2. Установить зависимости
pip install -r requirements.txt

# 3. Запустить приложение
streamlit run app.py
```

## 📁 Структура проекта

```
club_charter_expert/
├── app.py                    # Streamlit-приложение
├── agents/
│   ├── agent1_extractor.py   # Агент 1: извлечение структуры
│   ├── agent2_compliance.py  # Агент 2: проверка соответствия
│   └── agent3_improver.py    # Агент 3: заключение
├── tools/
│   ├── file_parser.py        # Парсинг PDF/DOCX
│   └── policy_checker.py     # База правил университета
├── utils/
│   ├── memory.py             # Память системы
│   └── demo_data.py          # Демонстрационные данные
└── requirements.txt
```

## 🔑 Используемые технологии

- **Claude claude-sonnet-4-20250514** через Anthropic API
- **Streamlit** — веб-интерфейс
- **pdfplumber / python-docx** — парсинг документов
- **JSON** — персистентная память

## 📋 Два сценария

**Сценарий 1 (условная задача НЕ срабатывает):**
Устав «Алматы Дебат Клуб» содержит все 10 обязательных разделов → Агент 2 не выявляет нарушений → система переходит напрямую к Агенту 3.

**Сценарий 2 (условная задача СРАБАТЫВАЕТ):**
Устав «CineMax Almaty» не содержит разделов «Этика», «Безопасность», «Отчётность», «Изменение устава» и имеет риски дискриминации → Агент 2 выявляет нарушения → автоматически запускается условная задача.
